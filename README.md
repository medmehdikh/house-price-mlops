# Projet house-prices-mlops

## Objectif
Ce projet est dédié pour appliquer les principes MLOps sur un projet ML.

## Sujet
Le projet consiste à préparer un modèle qui prédit les prix des maisons étant données ses caractéristiques.

## Jeu des données
Le jeu de données utilisé est disponible sur : https://www.kaggle.com/c/house-prices-advanced-regression-techniques
